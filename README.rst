===============
wagtailpolls
===============

A plugin for Wagtail that provides polling functionality
`Documentation on ReadTheDocs <https://wagtailpolls.readthedocs.org/en/latest/>`_

Installing
==========

Install using pip::

    pip install wagtailpolls

It works with Wagtail 1.0b2 and upwards.
